# Shopping Lists Manager

Веб-приложение для совместного управления списками покупок с real-time обновлениями.

## Технологии

**Frontend:**

- React
- TypeScript
- Vite

**Backend:**

- NestJS
- PostgreSQL
- WebSocket
- JWT Authentication

## Структура проекта

```
shopping-lists-app/
├── backend/          # NestJS API
└── frontend/         # React приложение
```
